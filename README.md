# 5G-Based-Vehicle-to-Vehicle-V2V-Communication-System

This project simulates a **secure Vehicle-to-Vehicle (V2V) communication system** over 5G.  
Vehicles exchange real-time data such as speed, distance, and multi-antenna (MIMO) signals.  

### Features
- **AI Collision Detection:** Predicts collision risk using `RandomForest` based on speed and distance.
- **AI Anomaly Detection:** Detects abnormal or suspicious telemetry with `IsolationForest`.
- **RSA Digital Signatures:** Ensures all messages are authentic and prevents spoofing.
- **Real-time Communication:** Built with Python, Flask, and Socket.IO for V2V messaging.
- **Multi-Antenna (MIMO) Simulation:** Improves reliability and speed of vehicle data exchange.
